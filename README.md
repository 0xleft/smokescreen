# Smokescreen

A pspy64 smoke screen that kills all pspy64 scanning threads

# Build

```bash
sudo apt install -y linux-headers-$(uname -r) xxd
make
```

# Run

```bash
sudo ./smokescreen # must run with root privileges
```
